# Game icons

The `icon` directory contains a master SVG for the game icon and platform-specific files generated from this vector file:

### `windows`
* `Icon.ico` is composed of PNGs ranging from 16x16px to 256x256px to accomodate all resolutions and hidpi setups. Used by the DirectX version.
* `Icon.bmp` is a 64x64 BMP for the OpenGL version. BMPs of higher resolutions result in very pixelated rendering, this size seems to be preferred by SDL2 on the platform.

### `linux`
* `Icon.png` is provided for convenience and backwards compatibility, since all major desktop environments support SVG, so the master icon file can be used directly in most cases.

### `macos`
* `Icon.icon` is an Apple Icon Composer project that overlays the game icon on a Liquid Glass background.
* `Icon.icns` has been generated from the Icon Composer with Xcode to support the expected resolutions by the platform.
* `Icon.bmp` is used by MonoGame at runtime and is a 256x256px version of the same Liquid Glass icon.

### `android`
* Adaptive icon resources as generated by Android Studio's Asset Studio. The icon foreground (`drawable/icon_foreground.xml`) is the vector representation of the icon, the background is defined as a color (`values/icon_background.xml`), and `mipmap-anydpi-v26` contains the glue to match them together.
